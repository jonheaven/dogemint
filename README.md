# Dogemint

Frontend library for minting Doginals and handling Dogecoin wallet operations. Designed for web3 dApps and browser extensions.

## Features
- Mint and inscribe Doginals
- Batch inscription support
- Dogecoin wallet management
- Configurable for node RPC or paid API services
- TypeScript support

## Getting Started
1. Install dependencies: `npm install`
2. Build: `npm run build`
3. Run tests: `npm test`

See the `docs/` folder for architecture, API, and provider guides.
